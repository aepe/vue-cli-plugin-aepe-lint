# vue-cli-plugin-aepe-lint

## 使用须知

> 本 vue-cli 插件 适用于 vscode 编辑器
>
> 1. 在 vue 项目中使用 `vue add vue-cli-plugin-aepe-lint` 安装组件。
> 2. 为了保证良好的使用条件 请在 vscode 上需要安装 `prettier` 插件

## 在 vuecli3.0+ 项目中安装

```bash
vue add vue-cli-plugin-aepe-lint
```

## 在 vuecli3.0+ 项目中的使用方式

### 格式化代码

```bash
yarn prettier
yarn eslint:fix
```
